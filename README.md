# Ellytaa Dessert

Prototype Website E-Commerce sebagai Proyek Akhir Mata Kuliah KAIT II Program Studi Administrasi Bisnis.

# Deskripsi Proyek

Ellytaa Dessert merupakan prototype website e-commerce yang dikembangkan sebagai proyek akhir Mata Kuliah KAIT II Program Studi Administrasi Bisnis. Website ini dibuat menggunakan HTML, CSS, dan JavaScript untuk mensimulasikan proses jual beli produk dessert secara online, mulai dari menampilkan katalog produk, menambahkan produk ke keranjang, wishlist, hingga proses checkout dan pencetakan struk.

Website dirancang dengan tampilan modern, responsif, serta mudah digunakan pada berbagai perangkat seperti komputer, laptop, tablet, maupun smartphone. Seluruh data website disimpan menggunakan LocalStorage, sehingga website tetap dapat digunakan tanpa memerlukan database.

# Tujuan Proyek

Tujuan pembuatan website ini adalah:

- Menerapkan konsep dasar e-commerce dalam bentuk website.
- Mengembangkan antarmuka website menggunakan HTML, CSS, dan JavaScript.
- Membuat tampilan website yang menarik, modern, dan responsif.
- Mengimplementasikan penyimpanan data menggunakan LocalStorage.
- Membuat simulasi transaksi pembelian dessert secara online.
- Memberikan pengalaman pengguna (User Experience) yang sederhana, nyaman, dan interaktif.

# Deskripsi Bisnis

Ellytaa Dessert merupakan toko online yang bergerak di bidang makanan penutup (dessert) dengan menyediakan berbagai pilihan cake, cookies, pudding, donut, pastry, tart, dan bread berkualitas premium.

Website ini dibuat untuk memudahkan pelanggan dalam melihat katalog produk, memilih produk favorit, menyimpan produk ke wishlist, melakukan pembelian secara online, serta mencetak bukti transaksi dengan mudah.

Ellytaa Dessert berkomitmen menghadirkan dessert berkualitas tinggi menggunakan bahan-bahan pilihan dengan cita rasa terbaik, harga yang terjangkau, dan pelayanan yang ramah.

# Analisis Pasar

Perkembangan teknologi digital telah mengubah kebiasaan masyarakat dalam melakukan pembelian makanan. Banyak pelanggan lebih memilih memesan makanan secara online karena lebih praktis, cepat, dan dapat dilakukan kapan saja.

Dessert menjadi salah satu produk kuliner yang memiliki permintaan tinggi, terutama untuk acara ulang tahun, perayaan, hadiah, maupun konsumsi sehari-hari. Oleh karena itu, Ellytaa Dessert memanfaatkan website e-commerce sebagai media pemasaran digital agar dapat menjangkau pelanggan lebih luas.

Ellytaa Dessert menerapkan model bisnis **Business to Customer (B2C)**, yaitu menjual produk dessert secara langsung kepada pelanggan melalui website.

# SEO dan Keamanan

Website dirancang menggunakan struktur HTML yang baik sehingga mudah dioptimalkan pada mesin pencari.
